# github_learn

> 个人代码基础 & 计算机视觉学习仓库  
> 目标：降低对 AI 的依赖，打牢编程和工程基础，留下清晰的成长记录。

---

## 1. 仓库用途

这个仓库用于记录我在以下几个方向上的学习与练习：

1. **编程基础**：以 Python 为主，后续会逐步补 C 语言等。
2. **算法与数据结构**：刷简单题，练习逻辑和代码表达。
3. **工程能力**：Git 使用、项目结构、脚本工具、调试与日志等。
4. **CV 相关实践**：和工作相关的小实验、小工具、小脚本。

同时，这个仓库也作为自己的“学习日志”与“成长档案”，方便以后回顾每一阶段自己在学什么、写过什么代码、踩过什么坑。

---

## 2. 我的当前阶段 & 目标

- 职业身份：**初入职场的计算机视觉算法工程师**
- 现状：
  - 写代码时对 AI（代码助手 / ChatGPT 等）的依赖非常高（> 95%）。
  - 能够完成工作中的一些任务，但基础知识不扎实，独立思考和实现能力需要加强。
- 近期目标（约 3–6 个月）：
  1. 不依赖 AI，也能写出日常的小脚本和工具。
  2. 看得懂项目中常见的模块/脚本，敢于重构和优化。
  3. 建立起自己的代码风格、调试习惯和工程意识。
